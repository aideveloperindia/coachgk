üé¨ ANIMATION SYSTEM
================================================================================

ANIMATION DURATIONS:

Fast:       200ms     Icon hovers, quick transitions
Standard:   300ms     Button hovers, card transitions
Medium:     500ms     Section entrance animations
Slow:       800ms     Hero entrance
Very Slow:  2000ms    Counter animations

EASING FUNCTIONS:

ease-out:       Default for entrance animations
ease-in-out:    For smooth bidirectional transitions
spring:         Natural bouncy feel (Framer Motion)

ANIMATION TYPES:

1. Scroll-Triggered Animations (useInView):
   - Entry Effect: fade-in-up
   - Initial State: opacity: 0, translateY: 20px
   - Final State: opacity: 1, translateY: 0
   - Duration: 600ms
   - Delay: Staggered (100ms per item)
   - Trigger: Once when section enters viewport
   - Margin: -100px (triggers before fully visible)

2. Hover Animations:
   - Cards: translate-y-[-4px], shadow-2xl
   - Buttons: scale-105, shadow-2xl
   - Icons: scale-110, rotate(5deg)
   - Links: translate-x-1, color change to accent

3. Auto-Animations:
   - Testimonials: Auto-rotate every 5000ms
   - Background Orbs: Infinite pulse (scale 1‚Üí1.2‚Üí1, 8-10s)
   - WhatsApp Button: Infinite ping effect
   - Scroll Indicator: Bounce up-down (20px, 2s)
   - Counter Numbers: Count from 0 to target (2s)

4. Click/Tap Animations:
   - Scale: scale-95 (press down effect)
   - Duration: 150ms

5. Page Load Animations:
   - Hero badge: scale 0.9‚Üí1, opacity 0‚Üí1
   - Hero heading: translateY 30px‚Üí0, opacity 0‚Üí1
   - Stagger delay: 200ms between elements

FRAMER MOTION VARIANTS:

Initial:    { opacity: 0, y: 50 }
Animate:    { opacity: 1, y: 0 }
Exit:       { opacity: 0, y: -50 }
Transition: { duration: 0.6, ease: "easeOut" }
