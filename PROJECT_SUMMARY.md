  primary: "#0A2540",  // Main brand color (navy)

}

## 🐛 Troubleshooting

### Website not loading?
```bash
# Kill the process and restart
npm run dev
```

### Images not showing?
- Ensure images are in `public/` folder
- Check file names match (case-sensitive!)
- Restart dev server

### "Module not found" error?
```bash
npm install
```

### Port 3000 in use?
```bash
npm run dev -- -p 3001
```
