Domain Setup Guide for gopisarvepalli.com
==========================================

Step 1: Set Up Email in GoDaddy
--------------------------------
1. Login to GoDaddy (godaddy.com)
2. Go to Email section from your account dashboard
3. Buy Email Hosting (if not already purchased):
   - Look for "Email" or "Workspace Email"
   - Buy email hosting plan (usually $2-5 per month per mailbox)
4. Create Email Addresses:
   - Create primary email: contact@gopisarvepalli.com
   - Can create additional emails like: info@gopisarvepalli.com, support@gopisarvepalli.com
5. Configure Email Forwarding (if needed):
   - Forward emails to personal Gmail or other email


Step 2: Connect Domain to Vercel
----------------------------------

Option A: Using Vercel Nameservers (Easiest)
---------------------------------------------
1. In Vercel Dashboard:
   - Go to your project settings
   - Navigate to "Domains"
   - Click "Add Domain"
   - Enter: gopisarvepalli.com
   - Also add: www.gopisarvepalli.com
   
2. Get Nameservers from Vercel:
   - Vercel will show you nameservers like:
     ns1.vercel-dns.com
     ns2.vercel-dns.com

3. Update in GoDaddy:
   - Go to GoDaddy Domain Settings
   - Find "Nameservers" section
   - Click "Change"
   - Select "Custom"
   - Add Vercel's nameservers
   - Save

Option B: Using A Records (Keep email working)
-----------------------------------------------
1. In Vercel Dashboard:
   - Go to "Domains" in project settings
   - Add domain: gopisarvepalli.com
   - Vercel will show DNS records

2. In GoDaddy DNS Settings:
   - Go to DNS Management
   - Add these records:
     Type: A
     Name: @
     Value: [Vercel's A record IP]
     TTL: 3600
     
     Type: CNAME
     Name: www
     Value: cname.vercel-dns.com
     TTL: 3600


Step 3: Verify & Test
-----------------------
1. Wait 1-24 hours for DNS propagation
2. Check if site loads: Visit gopisarvepalli.com
3. Check if www works: Visit www.gopisarvepalli.com
4. Test email: Send a test email to contact@gopisarvepalli.com


Step 4: SSL Certificate (FREE!)
--------------------------------
IMPORTANT: DO NOT BUY SSL FROM GODADDY!

- Vercel provides FREE SSL certificates automatically
- SSL is auto-enabled once DNS propagates
- No additional cost or setup needed
- Your site will have https://gopisarvepalli.com automatically
- Save money: Don't buy SSL certificates from GoDaddy

- Update: Email addresses in footer/contact sections

Troubleshooting
---------------
Email not working?
- Wait 24-48 hours after setup
- Check GoDaddy email settings
- Verify MX records are correct

Site not loading?
- Check if DNS records are correct
- Clear browser cache
- Use DNS checker: whatsmyip.org/dns-lookup/

Both not working?
- Use Option B (A Records) instead of nameservers
- This keeps GoDaddy in control of DNS for email
